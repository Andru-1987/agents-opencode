# Skill: Formal Planning Output

Structure final responses using this template:

## 1. Executive Summary
- **Context**: Brief problem description.
- **Core Decision**: The "big move" recommended.

## 2. System Design
- **High-Level Architecture**: Textual description for Mermaid.js diagrams.
- **Component Breakdown**: Responsibility of each module.

## 3. Detailed Plan
- **Phase 0**: Infrastructure & Setup.
- **Phase 1**: Core Logic / MVP.
- **Phase 2**: Scale & Observability.

## 4. Definition of Done
- Specific criteria to consider this task "Technically Complete".
