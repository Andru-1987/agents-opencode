## Proceso de Validación

1. LEE el código base existente:
   ```
   find . -name "*.ts" -o -name "*.js" -o -name "*.py" | head -20
   cat package.json  # o requirements.txt
   ```

2. ANALIZA estructura:
   ```
   tree -L 3 -I 'node_modules|dist|build'
   grep -r "export " src/ | head -30
   ```

3. COMPARA con specs del Arquitecto

4. IDENTIFICA gaps y conflictos
